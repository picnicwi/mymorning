สวัสดีตอนเช้าครับ ยินดีต้อนรับสู่ MorningInsight พอดแคสต์สรุปข่าว AI และเทคโนโลยี ประจำวันพฤหัสบดีที่ ยี่สิบสาม เมษายน สองพันห้าร้อยหกสิบเก้า วันนี้เรามีข่าวที่น่าสนใจครบทั้งห้าหมวดเลยครับ ตั้งแต่การแข่งขันโมเดล AI ที่เดือดขึ้นทุกวัน ไปจนถึงการนำ AI เข้าสู่ระบบธนาคารอย่างจริงจัง ฟังให้จบแล้วคุณจะรู้ว่าวงการเทคโนโลยีกำลังเดินไปทางไหนครับ


หมวดที่หนึ่ง AI และ Generative AI

เริ่มกันที่เรื่องแรกครับ จาก MIT Tech Review รายงานว่าล่าสุดถึงเดือนมีนาคม สองพันยี่สิบหก Anthropic ขึ้นมาครองอันดับหนึ่งในการจัดอันดับโมเดล AI โลก ตามมาด้วย xAI ของ Elon Musk แล้วก็ Google และ OpenAI ที่คุ้นเคยกันดี น่าสนใจตรงที่โมเดลจากจีนอย่าง DeepSeek และ Alibaba ก็ไล่กวดมาใกล้ชิดมากขึ้นเรื่อยครับ สถานการณ์แบบนี้หมายความว่าองค์กรที่กำลังเลือกเครื่องมือ AI ต้องติดตามการเปลี่ยนแปลงถี่ขึ้นกว่าเดิมมากครับ

เรื่องที่สองครับ นักวิจัยและบริษัทเทคโนโลยีกำลังพัฒนาทีม AI agent ที่ทำงานร่วมกันเป็นกลุ่ม เพื่อจัดการกับงานที่ซับซ้อนเกินกว่าโมเดลตัวเดียวจะรับมือได้ มีการพูดถึง AI co-scientist ที่ช่วยนักวิจัยทำงานวิทยาศาสตร์ได้อย่างรวดเร็วขึ้น การเปลี่ยนจาก AI ผู้ช่วยคนเดียว มาเป็นทีม agent ที่ประสานงานกันนั้น จะเปลี่ยนรูปแบบการทำงานขององค์กรในยุคถัดไปอย่างมากทีเดียวครับ

เรื่องที่สามครับ PwC เพิ่งเผยแพร่ผล AI Performance Study ประจำปี สองพันยี่สิบหก พบว่ามีการแยกทิศทางอย่างชัดเจน บริษัทที่ทำ AI ได้ผลดีมีกลุ่มเล็กๆ ที่วิ่งนำหน้าทิ้งห่างคนอื่นมาก ส่วนบริษัทส่วนใหญ่ยังตามหลัง กลุ่มที่นำอยู่มองว่า AI คือเครื่องมือสร้างการเติบโต ไม่ใช่แค่ช่วยประหยัดเวลา ถ้าองค์กรยังมองว่า AI เป็นแค่โปรแกรมช่วยงานทั่วไป ก็เสี่ยงจะตกขบวนในระยะยาวครับ


หมวดที่สอง Tech Trends แนวโน้มเทคโนโลยี

เรื่องแรกของหมวดนี้ครับ ในงาน Google Cloud Next สองพันยี่สิบหก Google ประกาศว่า Gemini Enterprise ตอนนี้สามารถขับเคลื่อนเครือข่าย agent อัตโนมัติที่เชื่อมต่อข้อมูล แอปพลิเคชัน และ workflow ทั่วทั้งองค์กรได้แล้ว Vertex AI ก็ได้รับการอัพเกรดให้รองรับสถาปัตยกรรม agent-first นี้ด้วย Google กำลังตั้งมาตรฐานใหม่ที่กดดันให้ AWS และ Azure ต้องเร่งพัฒนาตามครับ

เรื่องที่สองครับ Dell Technologies จับมือกับ NVIDIA เปิดตัว Dell AI Data Platform แพลตฟอร์มนี้ออกแบบมาเพื่อช่วยองค์กรค้นหาและนำข้อมูลมาใช้กับ AI และ autonomous agent ในระดับใหญ่ ทำงานบน infrastructure ขององค์กรเองได้เลย สำหรับองค์กรที่ต้องการความเป็น data sovereignty และความหน่วงต่ำ แพลตฟอร์มแบบนี้จะทำให้การนำ AI มาใช้ที่บ้านตัวเองง่ายขึ้นมากครับ

เรื่องที่สามครับ NVIDIA เปิดตัวแพลตฟอร์ม AI รุ่นใหม่ในชื่อรหัส Vera Rubin ซึ่งพัฒนาขึ้นมาเพื่อรองรับโมเดลที่มีขนาดระดับล้านล้านพารามิเตอร์โดยเฉพาะ ทั้งพลังประมวลผลและแบนด์วิธหน่วยความจำดีขึ้นอย่างก้าวกระโดด ความเป็นผู้นำด้านฮาร์ดแวร์ของ NVIDIA นั้นยังคงกำหนดทิศทางว่าการฝึกโมเดล AI ขนาดใหญ่จะมีต้นทุนและความเป็นไปได้แค่ไหนในอนาคตครับ


หมวดที่สาม Banking และ Financial Services

มาถึงหมวดที่น่าตื่นเต้นมากสำหรับวงการธนาคารครับ เรื่องแรก จากรายงานของ Finastra และ Fintech Global พบว่า ร้อยละเจ็ดสิบแปดของธนาคารทั่วโลกกำลังนำ generative AI ไปใช้งานจริงในปี สองพันยี่สิบหกแล้ว เทียบกับแค่ร้อยละแปดในปี สองพันยี่สิบสี่ เพิ่มขึ้นเกือบสิบเท่าภายในสองปีครับ และมีถึงร้อยละห้าสิบแปดที่ deploy AI ในการใช้งานจริงอย่างน้อยหนึ่งฟังก์ชันแล้ว AI ในธนาคารข้ามจุดแบ่งจากการทดลองมาเป็นสิ่งจำเป็นในการดำเนินธุรกิจแล้วครับ

เรื่องที่สองครับ ถ้า generative AI ขึ้นมาแพร่หลายแล้ว agentic AI ที่ระบบตัดสินใจและกระทำแทนมนุษย์นั้นยังอยู่ในช่วงเริ่มต้น มีเพียงร้อยละสามสิบเอ็ดของธนาคารที่เริ่มนำไปใช้ แต่ตัวเลขนี้กำลังขยายตัวเร็วมาก ธนาคารที่เดินหน้าก่อนจะได้เปรียบอย่างมากในการลดต้นทุนงาน back-office และ compliance ซึ่งมีมูลค่ามหาศาลครับ

เรื่องที่สามครับ McKinsey และ PwC รายงานตรงกันว่า ร้อยละเจ็ดสิบของธนาคารพาณิชย์นำ AI ไปใช้ในฟังก์ชันหลักอย่างน้อยหนึ่งด้านแล้ว และในบรรดาที่ลงทุนไป ร้อยละเจ็ดสิบแปดรายงานว่าได้ผลตอบแทนเป็นบวกภายในสิบแปดเดือน McKinsey ประมาณว่า generative AI จะสร้างมูลค่าให้วงการธนาคารในระดับที่น่าตื่นตาครับ ตอนนี้ข้อมูล ROI ที่ชัดเจนเหล่านี้ทำให้ธนาคารไม่มีข้ออ้างที่จะรอดูต่อไปแล้วครับ


หมวดที่สี่ Data Platforms แพลตฟอร์มข้อมูล

เรื่องแรกครับ Microsoft Fabric ผ่านมาสองปีครึ่งนับจากเปิดตัวอย่างเป็นทางการ ตอนนี้มีลูกค้าแล้วกว่าสามหมื่นหนึ่งพันราย และยังเป็นแพลตฟอร์มข้อมูลที่เติบโตเร็วที่สุดในประวัติศาสตร์ของ Microsoft ในงาน FabCon สองพันยี่สิบหก มีการนำเสนอ Fabric IQ ซึ่งเปลี่ยน Fabric จากแค่แพลตฟอร์มรวมข้อมูล ให้กลายเป็นรากฐานสำหรับ AI reasoning ที่เชื่อม agent ทั่วทั้งองค์กรได้ครับ

เรื่องที่สองครับ Databricks ประกาศ general availability ของ Lakebase บริการ Postgres แบบ serverless ที่สร้างอยู่ภายใน lakehouse โดยตรง ออกแบบมาเพื่อเป็น transactional system สำหรับ agent AI โดยเฉพาะ นี่คือการอุดช่องโหว่สำคัญมากครับ เพราะก่อนหน้านี้ AI agent ยังขาดระบบจัดการข้อมูล transactional ที่น่าเชื่อถือภายใน lakehouse

เรื่องที่สามครับ มีข่าวดีสำหรับองค์กรที่ใช้ทั้ง Microsoft Fabric และ Databricks พร้อมกันครับ ตอนนี้ Azure Databricks Unity Catalog สามารถอ่านข้อมูลจาก OneLake ได้แบบ native เข้าสู่ public preview แล้ว และ Snowflake interoperability ก็ถึง general availability แล้วด้วย การบรรจบกันของระบบนิเวศทั้งสองช่วยลดการซ้ำซ้อนของข้อมูลและความซับซ้อนในการดูแลระบบลงได้มากครับ


หมวดที่ห้า Microsoft AI

มาถึงหมวดสุดท้ายครับ เรื่องแรก Microsoft ประกาศ rebranding จาก Azure AI Foundry เป็น Microsoft Foundry ในเดือนมกราคม สองพันยี่สิบหก สัญญาณนี้บอกว่า Microsoft มองแพลตฟอร์มนี้ใหญ่กว่า Azure อย่างเดียวแล้ว และที่น่าสนใจมากคือตอนนี้ Microsoft Foundry เพิ่ม Claude จาก Anthropic เข้ามาให้เลือกใช้แล้ว องค์กรจึงมีตัวเลือกโมเดลหลากหลายขึ้นในแพลตฟอร์มที่ไว้ใจได้ และลดความเสี่ยงจากการผูกติดกับ vendor เดียวครับ

เรื่องที่สองครับ ในงาน Build สองพันยี่สิบหก Microsoft ประกาศว่า RAG หรือ Retrieval-Augmented Generation นั้น plug-and-play บน Azure แล้ว นักพัฒนาสามารถนำข้อมูลองค์กรมาใส่ได้เลยและได้คำตอบที่อิงข้อมูลจริง ลด hallucination ลงได้มาก ข่าวนี้สำคัญมากสำหรับสถาบันการเงินที่มีข้อมูลละเอียดอ่อนและไม่อยากส่งไปประมวลผลที่อื่นครับ

เรื่องสุดท้ายครับ Microsoft เปิดตัว Azure HorizonDB ฐานข้อมูลที่ออกแบบมาสำหรับ AI โดยเฉพาะ มี vector indexing ในตัว เพื่อรองรับการดึงข้อมูลและ memory ของระบบ agentic AI ในระดับองค์กร การมีฐานข้อมูลแบบนี้จาก Microsoft โดยตรงหมายความว่าองค์กรไม่ต้องไปหา third-party vector database อีกต่อไป ลดความซับซ้อนและต้นทุนลงได้อีกขั้นครับ


ห้าสิ่งที่ต้องจับตามองวันนี้

หนึ่ง GenAI ในธนาคารข้ามจุดเปลี่ยนแล้ว จากแปดเปอร์เซ็นต์มาเป็นเจ็ดสิบแปดเปอร์เซ็นต์ในสองปี ถือว่าเร็วมาก

สอง Agentic AI คือสมรภูมิการแข่งขันถัดไป ธนาคารที่เริ่มก่อนจะได้เปรียบด้านประสิทธิภาพอย่างมีนัยสำคัญ

สาม ทีม AI หลายตัวที่ทำงานร่วมกันกำลังเปลี่ยนนิยามของการ automate งานความรู้ในองค์กร

สี่ Microsoft Fabric และ Databricks กำลังมาบรรจบกันบน OneLake ลดภาระองค์กรที่ใช้ทั้งสองระบบ

ห้า ข้อมูล ROI ที่พิสูจน์ได้แล้วในระดับองค์กรขนาดใหญ่ ลบข้ออ้างสุดท้ายของการชะลอการลงทุน AI ออกไปแล้วครับ
