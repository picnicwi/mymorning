สวัสดีตอนเช้าครับ ยินดีต้อนรับสู่ MorningInsight พอดแคสต์สรุปข่าว AI และเทคโนโลยี ประจำวันพุธที่สิบห้า เมษายน สองพันห้าร้อยหกสิบเก้า วันนี้เรามีข่าวน่าสนใจครบทั้งห้าหัวข้อ ทั้งเรื่อง AI และ Generative AI แนวโน้มอุตสาหกรรมเทคโนโลยี ธนาคารและการเงิน แพลตฟอร์มข้อมูล และ Microsoft AI มาเริ่มกันเลยครับ


หัวข้อที่หนึ่ง AI และ Generative AI

เรื่องแรกจาก PwC ที่เพิ่งออกรายงาน AI Performance Study ประจำปีสองพันยี่สิบหก ผลการศึกษาพบว่า บริษัทเพียงยี่สิบเปอร์เซ็นต์กำลังกอบโกยผลประโยชน์ทางเศรษฐกิจจาก AI ถึงเจ็ดสิบห้าเปอร์เซ็นต์ของทั้งหมด ความแตกต่างคือบริษัทที่ประสบความสำเร็จไม่ได้ใช้ AI เพื่อลดต้นทุนอย่างเดียว แต่ใช้เพื่อสร้างรายได้ใหม่และโมเดลธุรกิจใหม่ด้วย ถ้าองค์กรไหนยังอยู่ในโหมดทดลองนำร่อง อาจเสียเปรียบเชิงโครงสร้างในระยะยาวครับ

เรื่องที่สองน่าสนใจมากครับ งานวิจัยที่ตีพิมพ์ใน Nature พบว่า AI Agent ที่ดีที่สุดในปัจจุบันยังสู้นักวิทยาศาสตร์มนุษย์ไม่ได้ในงานวิจัยที่ซับซ้อน AI Index Report สองพันยี่สิบหกระบุว่า AI ปรากฏในงานตีพิมพ์วิทยาศาสตร์ธรรมชาติราวหกถึงเก้าเปอร์เซ็นต์ แต่ Agent ยังขาดทั้งความลึกในการให้เหตุผลและการตัดสินเชิงทดลอง สิ่งที่ทำได้ดีคือการช่วยเสริมนักวิจัย ไม่ใช่แทนที่ครับ

เรื่องที่สาม ความกระตือรือร้นต่อ AI กำลังเริ่มเย็นลงจากจุดสูงสุด นักลงทุนเริ่มตั้งคำถามถึงการลงทุนด้านฮาร์ดแวร์และ ROI ที่ช้ากว่าที่คาด แต่ในอีกด้านหนึ่ง Micron ประกาศลงทุนหนึ่งแสนล้านดอลลาร์เพื่อสร้างโรงงานผลิตเซมิคอนดักเตอร์ที่ใหญ่ที่สุดในสหรัฐฯ ที่รัฐนิวยอร์ก ฐานโครงสร้างพื้นฐานในระยะยาวยังคงแข็งแกร่งครับ

เรื่องสุดท้ายของหัวข้อนี้ ผลสำรวจของ Gallup พบว่าพนักงานจำนวนมากยังต้านการใช้ AI แม้นายจ้างจะสั่งให้ใช้ก็ตาม สาเหตุมาจากความไม่ไว้วางใจ ขั้นตอนการทำงานที่ไม่ชัดเจน และการฝึกอบรมที่ไม่เพียงพอ นี่เตือนใจว่าการ deploy เทคโนโลยีอย่างเดียวไม่พอ ต้องลงทุนด้าน change management และการพัฒนาทักษะด้วยครับ


หัวข้อที่สอง แนวโน้มอุตสาหกรรมเทคโนโลยี

Google เปิดตัว Gemini สามจุดหนึ่ง Pro ให้ใช้งานทั่วโลกในต้นเดือนเมษายน โดยมุ่งเน้นการให้เหตุผลที่ซับซ้อนสำหรับการเขียนโค้ดและวิเคราะห์ข้อมูล ราคาอยู่ที่สองดอลลาร์ต่อหนึ่งล้าน input token และสิบสองดอลลาร์ต่อหนึ่งล้าน output token พร้อม context window ขนาดหนึ่งล้าน token แพลตฟอร์ม Gemini มีผู้ใช้งานแล้วเจ็ดร้อยห้าสิบล้านคน แสดงถึงความได้เปรียบในเรื่องฐานผู้ใช้ขนาดใหญ่ครับ

ServiceNow ได้ขยายแพลตฟอร์ม AI ร่วมกับพาร์ตเนอร์อย่าง Qlik DXC Technology BigPanda KODIS Holdings และ TrustCloud โดยมีเป้าหมายให้ AI Platform ของ ServiceNow เป็นศูนย์กลางการ orchestration ระดับองค์กร เชื่อมต่อ data fabric เครื่องมือความปลอดภัย และ supply chain เข้าด้วยกันในเวิร์กโฟลว์ที่ตรวจสอบได้ครับ

Nutanix เปิดตัว Nutanix Cloud Platform รุ่นใหม่ที่งาน NEXT สองพันยี่สิบหก ที่ชิคาโกเมื่อต้นเดือนเมษายน รองรับทั้ง VM container และ AI workload พร้อมกัน นอกจากนี้ยังเปิดตัวบริการ agentic AI ที่มุ่งเป้าไปยัง neocloud ซึ่งเป็นผู้ให้บริการคลาวด์เฉพาะด้าน AI รายใหม่ โดยเน้นการกำกับดูแลข้อมูลในสภาพแวดล้อม hybrid ครับ

รายงานของ Constellation Research เดือนเมษายนสองพันยี่สิบหกชี้ว่า AI ระดับองค์กรกำลังเปลี่ยนจากการทดลองไปสู่การ deploy จริงแล้ว แต่หลายองค์กรกำลังค้นพบช่องว่างสำคัญด้านโครงสร้างต้นทุน governance และสถาปัตยกรรม ความล้มเหลวในปีนี้ไม่ได้มาจากโมเดล AI แต่มาจาก governance ที่ไม่พร้อมครับ


หัวข้อที่สาม ธนาคารและบริการทางการเงิน

ข้อมูลอุตสาหกรรมปีสองพันยี่สิบหกยืนยันว่า ธนาคารห้าสิบแปดเปอร์เซ็นต์ได้ deploy Generative AI จริงจังในฟังก์ชันหลักอย่างน้อยหนึ่งอย่างแล้ว เพิ่มขึ้นจากสี่สิบห้าเปอร์เซ็นต์ในปีสองพันยี่สิบสาม ที่น่าทึ่งกว่าคือธนาคารเจ็ดสิบแปดเปอร์เซ็นต์กำลังนำ GenAI ไปใช้จริง ซึ่งเพิ่มขึ้นเกือบสิบเท่าจากแค่แปดเปอร์เซ็นต์เมื่อปีก่อน พื้นที่ที่ใช้มากที่สุดคือ underwriting การวิเคราะห์ความเสี่ยง และการบริการลูกค้าครับ

ผลวิจัยใหม่แสดงว่า ผู้นำด้าน compliance และความเสี่ยงในธนาคารถึงแปดสิบสองเปอร์เซ็นต์คาดว่าจะเพิ่มการลงทุน AI มากกว่ายี่สิบห้าเปอร์เซ็นต์ในช่วงสองถึงสามปีข้างหน้า และสี่สิบหกเปอร์เซ็นต์เน้นที่การลงทุน Generative AI โดยเฉพาะ แรงขับเคลื่อนหลักคือการทำงาน compliance อัตโนมัติ การวิเคราะห์ความเสี่ยงแบบเรียลไทม์ และการกำกับดูแล model risk ครับ

Large language model และ multimodal AI กำลังถูกผนวกเข้าสู่ระบบ payments และ retail banking เพื่อเร่งการตรวจจับการทุจริต ปรับปรุงกระบวนการ KYC และสนับสนุนการบริการลูกค้าแบบเรียลไทม์ผ่านอินเทอร์เฟซแชทและเสียงอัจฉริยะ ธนาคารกำลังก้าวข้ามการทดลองขนาดเล็กสู่การ deploy ในวงกว้างที่มีมาตรฐานและ governance ที่ชัดเจนครับ

หน่วยงานกำกับดูแลการเงินของสหรัฐฯ อย่าง FDIC และ Federal Reserve กำลังเร่งพัฒนากรอบนวัตกรรม AI เพื่อตามทันการนำ AI มาใช้ในธนาคารที่รวดเร็วขึ้น โดยเน้นข้อกำหนดด้าน explainability แนวทาง model risk management และกำแพงคุ้มครองผู้บริโภค เนื่องจาก AI เริ่มมีบทบาทในการตัดสินใจด้านสินเชื่อและ compliance ครับ


หัวข้อที่สี่ แพลตฟอร์มข้อมูล

ที่งาน FabCon สองพันยี่สิบหก Microsoft และ Databricks ประกาศการรองรับการอ่านข้อมูลแบบ native จาก OneLake ผ่าน Azure Databricks Unity Catalog ซึ่งอยู่ใน public preview แล้ว นอกจากนี้ Azure Databricks จะรองรับการเขียนและจัดเก็บข้อมูลใน OneLake โดยตรง โดยไม่ต้องใช้ทรัพยากร storage เพิ่ม นับเป็นการเชื่อมต่อ lakehouse ที่แน่นแฟ้นขึ้นมากและลดต้นทุนโครงสร้างพื้นฐานครับ

Azure Databricks ประกาศชุดความสามารถใหม่ที่งาน FabCon สองพันยี่สิบหก ได้แก่ Lakebase ที่เข้าสู่ general availability แล้ว Lakeflow Connect ที่เพิ่ม free tier เพื่อลดอุปสรรคการรับข้อมูล และ Genie ที่เพิ่ม AI analytics ด้วยภาษาธรรมชาติเข้ามาในแพลตฟอร์ม รวมถึงการผนวกรวมกับ Microsoft สามร้อยหกสิบห้าที่ช่วยส่งข้อมูลเชิงลึกให้ถึงมือผู้ใช้งานจริงได้ง่ายขึ้นครับ

งาน FabCon และ SQLCon สองพันยี่สิบหกของ Microsoft มุ่งเน้นวิสัยทัศน์การรวม database และ Microsoft Fabric เข้าเป็นแพลตฟอร์มข้อมูลเดียว เป้าหมายคือขจัดขอบเขตระหว่าง operational database และ analytical data lakehouse โดยฝัง Azure SQL เข้าไปใน Fabric โดยตรง เพื่อลดความซับซ้อนทางสถาปัตยกรรมสำหรับทีมข้อมูลระดับองค์กรครับ


หัวข้อที่ห้า Microsoft AI

Microsoft ประกาศให้ multi-agent orchestration ใน Copilot Studio พร้อมใช้งานทั่วไปแล้วสำหรับลูกค้าที่มีสิทธิ์ทั้งหมดตั้งแต่เดือนเมษายนสองพันยี่สิบหก Agent สามารถทำงานเป็นระบบที่ประสานกัน ดังที่เห็นใน Ask Microsoft web agent ซึ่ง sub-agent ที่เชี่ยวชาญเฉพาะด้านหลายตัวถูก orchestrate โดย master coordinator ปรับปรุงครอบคลุมช่องทางเสียง การออกแบบ workflow และประสบการณ์ผู้สร้างครับ

Microsoft ประกาศ Microsoft สามร้อยหกสิบห้า E เจ็ด หรือ Frontier Suite ที่จะเปิดตัว วันที่หนึ่ง พฤษภาคม สองพันยี่สิบหก แพ็กเกจนี้รวม Microsoft สามร้อยหกสิบห้า E ห้า สำหรับ productivity ที่ปลอดภัย Microsoft Entra Suite สำหรับ identity Microsoft สามร้อยหกสิบห้า Copilot สำหรับ AI ในการทำงาน และ Microsoft Agent สามร้อยหกสิบห้าซึ่งเป็น control plane สำหรับกำกับดูแลและขยาย AI agent ทั่วทั้งองค์กรครับ

Microsoft สองพันยี่สิบหก Release Wave หนึ่ง ครอบคลุมเดือนเมษายนถึงกันยายน สองพันยี่สิบหก ถือเป็นการเปลี่ยนแปลงครั้งสำคัญของ Copilot จาก intelligent assistant ไปเป็น autonomous agentic platform ความสามารถหลักรวมถึง multi-agent orchestration ใน Copilot Studio การกำกับดูแล AI ด้วย agent Work IQ สำหรับ Dataverse intelligence การรองรับ Model Context Protocol หรือ MCP server และระบบจัดการ Copilot credit ที่ละเอียดขึ้นครับ


ห้าสิ่งที่ต้องจับตามองวันนี้

หนึ่ง ช่องว่างผลตอบแทน AI กว้างขึ้นเรื่อยๆ องค์กรที่ยังอยู่ในโหมดทดลองเสี่ยงถูกทิ้งห่างอย่างถาวร

สอง Google Gemini สามจุดหนึ่ง Pro กำลังสร้างแรงกดดันด้านราคาในตลาด AI model ทั้งอุตสาหกรรมต้องปรับกลยุทธ์

สาม ธนาคารและสถาบันการเงินเป็นอุตสาหกรรมที่เดินหน้านำ AI ไปใช้จริงเร็วที่สุด โดยเฉพาะด้าน compliance และ fraud detection

สี่ Microsoft กำลังรวมทุกอย่างเป็นแพลตฟอร์มเดียว ทั้ง data และ AI agent ผ่าน Fabric และ Copilot Studio ธุรกิจที่ใช้ Microsoft stack ควรติดตามอย่างใกล้ชิด

ห้า การต่อต้าน AI จากพนักงานเป็นความเสี่ยงที่ซ่อนอยู่ในทุกองค์กร การ deploy เทคโนโลยีต้องคู่กับการลงทุนในคน


ขอบคุณที่รับฟัง MorningInsight ครับ หวังว่าข่าวสารวันนี้จะเป็นประโยชน์สำหรับการทำงานและการตัดสินใจของคุณ มีวันที่ดีและขับรถปลอดภัยนะครับ พบกันพรุ่งนี้เช้า
