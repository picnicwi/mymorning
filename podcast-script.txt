สวัสดีตอนเช้าครับ ยินดีต้อนรับสู่ MorningInsight พอดแคสต์สรุปข่าว AI และเทคโนโลยี ประจำวันอังคารที่ ยี่สิบเอ็ด เมษายน สองพันห้าร้อยหกสิบเก้า ครับ

วันนี้เรามีข่าวสาร สิบหก เรื่อง จาก ห้า หมวดหมู่สำคัญ ทั้ง AI และ Generative AI เทรนด์อุตสาหกรรมเทค ธนาคารและการเงิน แพลตฟอร์มข้อมูล และ Microsoft AI ครับ มาเริ่มกันเลยนะครับ


หมวดแรก AI และ Generative AI

เริ่มต้นกันด้วยข่าวใหญ่จากวงการ AI ครับ Stanford AI Index ล่าสุดรายงานว่า Anthropic ขึ้นแท่นผู้นำการจัดอันดับโมเดล AI ระดับโลก ณ มีนาคม ปีนี้ ตามมาด้วย xAI ของ Elon Musk, Google และ OpenAI ตามลำดับ ที่น่าสนใจคือโมเดลจากจีน อย่าง DeepSeek ก็ติดอันดับเข้ามาด้วยครับ สิ่งที่ต้องตระหนักคือ ความสามารถของ AI กำลังกลายเป็น commodity มากขึ้นเรื่อยๆ บริษัทที่จะได้เปรียบต้องแข่งกันที่การนำ AI ไปใช้งานจริงให้ดี ไม่ใช่แค่เลือกโมเดลที่ดีที่สุดอีกต่อไปครับ

ข่าวต่อมา startup เครื่องมือเขียนโค้ด AI อย่าง Cursor กำลังเจรจาระดมทุน สองพันล้านดอลลาร์สหรัฐ ที่มูลค่าบริษัทสูงกว่า ห้าหมื่นล้านดอลลาร์ ข้อมูลนี้มาจาก CNBC ครับ ตัวเลขนี้บอกชัดเลยว่า developer tools ที่ขับเคลื่อนด้วย AI กลายเป็นหนึ่งในลำดับความสำคัญสูงสุดของการลงทุนองค์กรในปีนี้ครับ

PwC เพิ่งเผยแพร่ผลการศึกษา AI Performance ปี สองพันยี่สิบหก พบว่ากลุ่มบริษัทชั้นนำ ยี่สิบเปอร์เซ็นต์ กวาดผลตอบแทนทางเศรษฐกิจจาก AI ไปถึง เจ็ดสิบห้าเปอร์เซ็นต์ ของทั้งหมด สำหรับบริษัทในอุตสาหกรรมการเงิน ข้อมูลนี้คือสัญญาณเตือนที่ชัดเจนว่า การรอช้าในการนำ AI มาใช้มีต้นทุนที่สูงมาก ไม่ใช่แค่เสียโอกาสครับ

และปิดหมวดนี้ด้วยงานวิจัยจากวารสาร Nature ที่บอกว่า นักวิทยาศาสตร์มนุษย์ยังคงทำผลงานได้ดีกว่า AI agents ในงานวิจัยที่ซับซ้อน แม้ AI จะเก่งในงานย่อยที่มีขอบเขตชัดเจน แต่ยังสู้มนุษย์ไม่ได้ในโจทย์ที่ต้องการการคิดเชิงลึกหลายขั้นตอน ข้อค้นพบนี้บอกว่า รูปแบบการทำงานร่วมกันระหว่างมนุษย์กับ AI ยังคงเป็น paradigm หลักในอุตสาหกรรมที่ใช้ความรู้เชี่ยวชาญสูงต่อไปอีกสักพักครับ


หมวดที่สอง เทรนด์อุตสาหกรรมเทค

OpenAI Codex ซึ่งเปิดตัวเมื่อปีที่แล้ว ปัจจุบันมีผู้ใช้งานต่อสัปดาห์ทะลุ สามล้านคนแล้วครับ จาก สองล้าน เมื่อไม่กี่สัปดาห์ก่อน Codex เป็น AI agent ที่เขียนโค้ดและแก้ไขระบบซอฟต์แวร์ได้แบบครบวงจร ตัวเลขการเติบโตนี้บ่งชี้ว่า autonomous coding agents กำลังปรับเปลี่ยน workflow ของทีมวิศวกรรมซอฟต์แวร์ในองค์กรอย่างจริงจังและรวดเร็วครับ

NVIDIA เปิดตัวแพลตฟอร์ม AI รุ่นใหม่ชื่อ Vera Rubin พร้อม GPU รุ่น H300 ที่มีประสิทธิภาพสูงกว่าเดิมอย่างมีนัยสำคัญ ทั้งในด้านพลังการประมวลผลและแบนด์วิดท์หน่วยความจำ แพลตฟอร์มนี้วางเป้าหมายไว้สำหรับองค์กรที่ต้องการ infrastructure สำหรับ AI ขนาดใหญ่ครับ ถ้าบริษัทของคุณกำลังวางแผน AI deployment ขนาดใหญ่ในปี สองพันยี่สิบเจ็ด ควรพิจารณา Vera Rubin ไว้ใน roadmap ด้วยครับ

นักวิเคราะห์อุตสาหกรรมรายงานว่า AI กำลังรวมตัวกับ Platform Engineering โดยเคลื่อนจากการใช้งานส่วนบุคคลไปสู่การประสานงาน workflow ทั้งฝ่ายงานในองค์กร Salesforce ได้บุกเบิก Agentic License Model ซึ่งเป็นการเปลี่ยนจากการคิดค่าบริการต่อ user ไปเป็นแบบ outcome-based นี่คือการเปลี่ยนแปลงพื้นฐานในวิธีที่องค์กรจะซื้อและกำหนดราคา AI ครับ ต้องจับตาดูว่า vendor รายอื่นจะตามมาแบบเดียวกันไหม


หมวดที่สาม ธนาคารและบริการการเงิน

ข้อมูลใหม่เปิดเผยว่า ห้าสิบแปดเปอร์เซ็นต์ ของธนาคารได้นำ Generative AI ไปใช้จริงในฟังก์ชันธุรกิจหลักอย่างน้อยหนึ่งอย่างแล้วในปีนี้ เทียบกับ แปดเปอร์เซ็นต์ ในปี สองพันยี่สิบสี่ ซึ่งถือว่าเพิ่มขึ้นอย่างก้าวกระโดดมากครับ นี่คือสัญญาณชัดเจนว่า GenAI ในภาคธนาคารได้ผ่านจุดเปลี่ยนจากการทดลองสู่การนำไปใช้จริงในระดับปฏิบัติการแล้วครับ

ผลสำรวจผู้บริหารด้านความเสี่ยงและ compliance ในธนาคาร พบว่า ร้อยละ แปดสิบสอง คาดว่าการลงทุนด้าน AI จะเติบโตมากกว่า ยี่สิบห้าเปอร์เซ็นต์ ในสองถึงสามปีข้างหน้า การลงทุนในระดับนี้จะเร่งการควบรวมกิจการของ vendor และยกระดับมาตรฐานสำหรับ technology partners ที่ให้บริการสถาบันการเงินครับ

FDIC ได้กล่าวสุนทรพจน์ชื่อ Innovation at the Speed of Markets อธิบายว่าหน่วยงานกำกับดูแลกำลังพัฒนา framework เพื่อตามทัน AI และ fintech ในภาคธนาคาร ความชัดเจนด้านกฎระเบียบจาก FDIC นี้ลดความไม่แน่นอนในการดำเนินงาน และถือเป็นสัญญาณไฟเขียวให้สถาบันการเงินที่รอคำแนะนำด้าน compliance ก่อนขยายโปรแกรม AI สามารถเดินหน้าต่อได้แล้วครับ


หมวดที่สี่ แพลตฟอร์มข้อมูล

ในงาน FabCon สองพันยี่สิบหก Azure Databricks ประกาศการเปิดตัวอย่างเป็นทางการของ Lakebase ซึ่งเป็น managed serverless Postgres service ที่สร้างอยู่ใน lakehouse โดยตรง ออกแบบมาเพื่อเป็นระบบ transactional สำหรับ AI agents Lakebase เชื่อมช่องว่างระหว่าง analytical lakehouse และ transactional database ในแพลตฟอร์มเดียว ซึ่งเป็นส่วนประกอบสำคัญสำหรับการสร้าง agentic AI applications ระดับ production ครับ

Microsoft และ Databricks ร่วมประกาศว่า การอ่านข้อมูลจาก OneLake ผ่าน Azure Databricks Unity Catalog เข้าสู่ public preview แล้วครับ ความสามารถนี้ช่วยให้ข้อมูล transactional และ analytical ไหลได้อย่างอิสระระหว่างสองแพลตฟอร์ม ลดความซับซ้อนของ data governance สำหรับองค์กรที่ใช้ทั้ง Microsoft Fabric และ Databricks ควบคู่กันครับ

Microsoft ยังใช้งาน FabCon และ SQLCon ปีนี้ประกาศกลยุทธ์รวม databases และ Microsoft Fabric เข้าไว้ในแพลตฟอร์มข้อมูลเดียวกัน การรวม SQL, analytics และบริการข้อมูล AI เข้าใน Fabric จะลดความซับซ้อนของ integration สำหรับทีม data ในองค์กร และเสริมความแข็งแกร่งของ Microsoft ในการแข่งขันกับ Snowflake และ Databricks ครับ


หมวดสุดท้าย Microsoft AI

Microsoft ได้เปลี่ยนชื่อ Azure AI Foundry เป็น Microsoft Foundry เป็นส่วนหนึ่งของกลยุทธ์ทำให้ agentic AI กลายเป็น layer มาตรฐานทั่วทั้งระบบ cloud, productivity และ security และที่น่าตื่นเต้นมากคือ Claude ของ Anthropic ตอนนี้พร้อมให้ใช้งานใน Microsoft Foundry แล้วครับ ซึ่งเปิดทางเลือกโมเดล AI ชั้นนำนอกเหนือจาก GPT-4 สำหรับองค์กรที่ใช้ Azure ecosystem

Microsoft เปิดตัว Azure HorizonDB ซึ่งเป็นฐานข้อมูลที่ออปติไมซ์สำหรับ AI มีความสามารถ vector indexing ในตัว เป็นส่วนหนึ่งของการอัปเดต Azure platform ประจำเดือนเมษายนนี้ครับ HorizonDB ช่วยลดความจำเป็นในการใช้ vector database แยกต่างหาก ทำให้สถาปัตยกรรมของ AI applications บน Azure เรียบง่ายและจัดการได้ง่ายขึ้นมากครับ

และปิดท้ายหมวดนี้ Microsoft ยังเพิ่ม Grok สี่จุดสอง และโมเดล MAI ใหม่เข้าไปใน Microsoft Foundry model catalog พร้อมขยาย Genie ไปครอบคลุมญี่ปุ่นและเกาหลีด้วยครับ การขยาย catalog โมเดลและครอบคลุมภูมิภาค Asia-Pacific นี้เร่งการนำ AI ไปใช้ในองค์กรและเปิดทางเลือกโมเดลให้กว้างขึ้นสำหรับลูกค้า Azure ทั่วโลกครับ


ห้าสิ่งที่ต้องจับตามองวันนี้

หนึ่ง Anthropic ขึ้นแท่นผู้นำ AI model rankings ระดับโลก สัญญาณว่าการแข่งขันโมเดล AI กำลัง commoditize และองค์กรต้องแข่งกันที่ความสามารถในการ implement ไม่ใช่แค่การเลือกโมเดลครับ

สอง GenAI ในธนาคารทะลุ ห้าสิบแปดเปอร์เซ็นต์ การรับเอา AI ในภาคการเงินก้าวข้ามจุดเปลี่ยนแล้ว บริษัทที่ยังไม่ลงมือจะเสียเปรียบมากขึ้นเรื่อยๆ ครับ

สาม Microsoft Foundry รองรับ Claude ของ Anthropic แล้ว ผู้ใช้ Azure มีตัวเลือก AI model ชั้นนำหลายตัวในที่เดียวกันครับ

สี่ Databricks Lakebase เปิดตัวอย่างเป็นทางการแล้ว ฐานข้อมูล Postgres ที่ออกแบบมาสำหรับ AI agents คือรากฐานสำคัญของ production agentic AI ครับ

ห้า PwC พบว่า ยี่สิบเปอร์เซ็นต์ ของบริษัทกวาด เจ็ดสิบห้าเปอร์เซ็นต์ ของผลตอบแทน AI ยิ่งช้ายิ่งเสียเปรียบ ถึงเวลาต้องเร่งลงมือจริงจังครับ
