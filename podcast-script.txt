สวัสดีตอนเช้าครับ ยินดีต้อนรับสู่ MorningInsight พอดแคสต์สรุปข่าว AI และเทคโนโลยี ประจำวันจันทร์ที่สิบสามเมษายน สองพันห้าร้อยหกสิบเก้า วันนี้เรามีข่าวที่น่าสนใจจากห้าหัวข้อหลัก ได้แก่ AI และ Generative AI เทรนด์อุตสาหกรรมเทค ธนาคารและการเงิน แพลตฟอร์มข้อมูล และ Microsoft AI ครับ มาเริ่มกันเลยดีกว่า

หัวข้อแรกครับ AI และ Generative AI

ข่าวแรกมาจาก Meta ครับ Meta เปิดตัวโมเดลใหม่ชื่อว่า Muse Spark หลังจากที่ดึงตัว Alexandr Wang จาก Scale AI เข้ามา สิ่งที่น่าสนใจคือ Muse Spark เป็นโมเดลขนาดกะทัดรัดที่ทำงานได้เทียบเท่า Llama สี่รุ่นเก่า แต่ใช้คอมพิวต์น้อยกว่าถึงสิบเท่า นี่บ่งบอกว่าเทรนด์ของ AI กำลังเปลี่ยนจากการแข่งกันที่ขนาดโมเดล มาเป็นการแข่งกันที่ประสิทธิภาพต่อการใช้ทรัพยากรครับ

ถัดมา NVIDIA ประกาศเปิดตัว AI Platform รุ่นใหม่ชื่อ Vera Rubin ที่งาน CES ปีนี้ แพลตฟอร์มนี้มาพร้อมกับการปรับปรุงด้านพลังประมวลผลและแบนด์วิดธ์ของหน่วยความจำอย่างก้าวกระโดด ซึ่งจะกำหนดทิศทางโครงสร้างพื้นฐาน AI ขององค์กรไปจนถึงปีสองพันยี่สิบเจ็ดเลยทีเดียวครับ

มีงานวิจัยที่น่าตื่นเต้นมากออกมาครับ นักวิจัยพัฒนาวิธีผสมผสาน Neural Network กับ Symbolic Reasoning ที่ลดการใช้พลังงานได้ถึงหนึ่งร้อยเท่า พร้อมกับความแม่นยำที่ดีขึ้นด้วย สำหรับวงการการเงิน นี่สำคัญมากเพราะจะทำให้ AI ทำงานบน Edge หรืออุปกรณ์พกพาได้โดยไม่เสียค่าใช้จ่ายสูงเหมือนเดิมครับ

ผลการศึกษากับกลุ่มตัวอย่างกว่าหนึ่งแสนคนพบว่า Generative AI สามารถทำคะแนนสูงกว่ามนุษย์โดยเฉลี่ยในการทดสอบความคิดสร้างสรรค์ได้แล้ว นี่เป็นสัญญาณว่า AI กำลังก้าวข้ามขอบเขตจากการเป็นเครื่องมือช่วยงาน ไปสู่การเป็นคู่แข่งในงานสร้างสรรค์ด้วยครับ

ในด้านการนำ AI เข้าองค์กร มีรายงานว่าเกือบหนึ่งในสามขององค์กรขนาดใหญ่ทั่วโลกจะมีการกำหนดบทบาทให้ AI Worker อย่างเป็นทางการภายในสิ้นปีนี้ ตัวอย่างที่ดีคือ AI ของ Klarna ทำงานได้เทียบเท่าพนักงานกว่าเจ็ดร้อยคน อย่างไรก็ตาม BCG เตือนว่าเจ็ดสิบเปอร์เซ็นต์ของความล้มเหลวในการนำ AI ไปใช้มาจากการขาด Context และการออกแบบกระบวนการที่ไม่ดี ไม่ใช่ตัวโมเดลครับ

สุดท้ายในหัวข้อนี้ โมเดล AI เฉพาะทางและ Multimodal Model กำลังได้รับความนิยมในองค์กรมากขึ้นเรื่อยๆ โมเดลที่เทรนบนข้อมูลเฉพาะอุตสาหกรรมมีความแม่นยำสูงกว่าและลด Hallucination ได้มาก ซึ่งมีความสำคัญมากสำหรับงาน Compliance และการตัดสินใจด้านสินเชื่อในธนาคารครับ

มาที่หัวข้อที่สองกันครับ เทรนด์อุตสาหกรรมเทค

องค์กรใหญ่หลายแห่งกำลังเปลี่ยนมุมมองเกี่ยวกับ AI ครับ จากการทดลองเล่นๆ มาเป็นการนำไปใช้งานจริงที่ต้องมีความปลอดภัยและผลตอบแทนที่วัดได้ ความเสี่ยงจาก Prompt Injection และการจัดการข้อมูลในประเทศกลายเป็นข้อกำหนดสำคัญก่อนนำ AI เข้าสู่ Production แล้วครับ

AI Agent กำลังเปลี่ยนโฉมการทำงานอย่างเห็นได้ชัด ปีนี้เราเห็นองค์กรที่ใช้ Agent ทำงานแทนทีมงานขนาดใหญ่ได้ บางองค์กรสามารถเปิดตัวแคมเปญการตลาดระดับโลกได้ภายในไม่กี่วัน แทนที่จะต้องใช้เวลาเป็นไตรมาส นี่คือการปรับโครงสร้างองค์กรครั้งใหญ่ที่กำลังเกิดขึ้นจริงครับ

Morgan Stanley คาดการณ์ว่าจะมีการลงทุนในโครงสร้างพื้นฐาน AI ทั่วโลกสูงถึงสามล้านล้านดอลลาร์ภายในปีสองพันยี่สิบแปด ครอบคลุมทั้ง Data Center พลังงาน ห่วงโซ่อุปทานชิป และซอฟต์แวร์เฉพาะ AI ตัวเลขนี้บ่งบอกว่า AI ได้ก้าวข้ามสู่การเป็นโครงสร้างพื้นฐานสำคัญเทียบเท่า Cloud Computing ในยุคต้นสองพันสิบแล้วครับ

Google DeepMind กำลังพัฒนา World Model แบบสามมิติแบบ Real Time ชื่อ Genie ในขณะที่หุ่นยนต์มนุษย์เริ่มเข้าสู่การใช้งานจริงแล้ว AI ยังเข้าช่วยในการค้นพบทางวิทยาศาสตร์ด้านฟิสิกส์ เคมี และชีววิทยา ซึ่งขยายขอบเขตของ AI ออกไปจากโลกดิจิทัลสู่โลกกายภาพครับ

มาถึงหัวข้อที่สำคัญมากสำหรับอุตสาหกรรมของเราครับ ธนาคารและการเงิน

หลังจากปีที่ผ่านมาที่หลายธนาคารยังอยู่ในช่วงทดลอง ตอนนี้เราเห็นการ Deploy AI สู่ Production Scale จริงๆ แล้ว ทั้งในด้าน Customer Engagement การตัดสินใจด้านสินเชื่อ และการดำเนินงานหลังบ้าน Oracle Financial Services รายงานว่าธนาคารหลายแห่งกำลังขยาย AI ไปทั่วระบบ Core Banking พร้อม Compliance Guardrails ที่ฝังอยู่ในระบบแล้วครับ

แต่มีข้อมูลที่น่าเป็นห่วงครับ Wolters Kluwer พบว่าสามสิบเอ็ดจุดแปดเปอร์เซ็นต์ของสถาบันการเงินมี AI ทำงานใน Production แล้ว แต่มีเพียงสิบสองจุดสองเปอร์เซ็นต์เท่านั้นที่มีกลยุทธ์ AI ที่ชัดเจนและมีทรัพยากรรองรับ ช่องว่างนี้คือความเสี่ยงด้าน Regulatory ที่กำลังสะสมอยู่ครับ

ในแง่ขนาดของผลกระทบ ผลสำรวจ Oracle Financial Services ชี้ว่าเจ็ดสิบเปอร์เซ็นต์ของผู้บริหารธนาคารมองว่า Agentic AI คือการเปลี่ยนแปลงระดับ Transformational ไม่ใช่แค่การปรับปรุงเล็กน้อย การประมาณการณ์ชี้ว่า AI จะสร้างมูลค่าให้วงการธนาคารโลกระหว่างสองร้อยพันล้านถึงสามร้อยสี่สิบพันล้านดอลลาร์ต่อปี และลดต้นทุนดำเนินการได้ถึงยี่สิบเปอร์เซ็นต์ครับ

ด้านนวัตกรรม การยื่นจดสิทธิบัตรที่เกี่ยวกับ AI ในภาคการเงินเพิ่มขึ้นสองร้อยห้าสิบเปอร์เซ็นต์ในห้าปีที่ผ่านมา Mastercard เปิดตัว AI Engine ของตัวเองที่เทรนบนข้อมูลธุรกรรมหลายสิบปี ซึ่งทำผลงานได้ดีกว่า Machine Learning ทั่วไปในการตรวจจับธุรกรรมที่หายากแต่ถูกกฎหมายครับ

อย่างไรก็ตาม ยังมีอุปสรรคสำคัญครับ ระบบ Legacy และ Compliance หนักๆ ยังคงเป็นตัวถ่วงความเร็วในการนำ AI ไปใช้ ผู้กำกับดูแลระบุว่า Explainability ความโปร่งใส และการจัดการ Bias เป็นความกังวลหลักที่ต้องแก้ไขตลอดปีนี้ครับ

หัวข้อที่สี่ครับ Data Platforms ทั้ง Databricks และ Microsoft Fabric

Databricks ประกาศ General Availability ของ Genie Code Agent Mode แล้ว ซึ่งทำให้ทีม Data Engineering สามารถมอบหมายงาน Pipeline ที่ซับซ้อนให้ AI ดูแลได้แบบ Autonomous หลายขั้นตอนโดยไม่ต้องคอยดูแล พร้อมกันนี้ Databricks Runtime สิบแปดจุดหนึ่งก็ GA แล้วด้วยครับ

Lakebase Instance ของ Databricks ตอนนี้เปิด Autoscaling และ Scale to Zero เป็นค่าเริ่มต้นแล้ว พร้อม Workspace Branching และ Instant Restore ลดภาระงานดูแลระบบไปได้มาก และยังมี Connector ที่เชื่อมต่อกับ Workday HCM โดยตรง สำหรับการนำข้อมูล HR เข้าสู่ระบบ Data Engineering ครับ

Microsoft Fabric มีอัปเดตสำคัญในเดือนมีนาคมครับ High Concurrency Notebook Sessions เพิ่มขึ้นเป็นห้าสิบ Session พร้อมกัน Enterprise JDBC Driver สำหรับ Spark SQL พร้อม GA และ OneLake Catalog Search API ใหม่ช่วยค้นหา Data Asset ข้าม Workspace ได้จาก API เดียว เหมาะมากสำหรับองค์กรขนาดใหญ่ที่มีโครงสร้าง Data Mesh ครับ

Eventstream ของ Fabric ก็ได้รับการอัปเกรดครั้งใหญ่เช่นกันครับ ทั้ง Connector ที่มากขึ้น การประมวลผล Stream แบบ Real Time ที่ดีขึ้น และเครือข่ายที่ปลอดภัยมากขึ้น พร้อมกับการเพิ่ม Identity Limit จากหนึ่งพันเป็นหนึ่งหมื่น ทำให้ Fabric กำลังกลายเป็น Platform Analytics Real Time ครบวงจรสำหรับองค์กรครับ

มาถึงหัวข้อสุดท้ายของวันนี้ครับ Microsoft AI และ Copilot

Microsoft 365 Copilot Wave สาม เปลี่ยนโฉม Copilot จาก Chat Assistant ธรรมดา มาเป็น AI Layer ที่ทำงานอัตโนมัติหลายขั้นตอนใน Word Excel และ PowerPoint ได้เลย มีฟีเจอร์ใหม่ทั้ง Inbox Awareness Calendar Awareness และ Meeting Video Recap ทำให้พนักงานประหยัดเวลาจัดการเอกสารได้มากขึ้นอย่างเห็นได้ชัดครับ

ข่าวสำคัญมากสำหรับองค์กรที่ใช้ Microsoft ครับ ตั้งแต่วันที่หนึ่งพฤษภาคม สองพันยี่สิบหก Microsoft 365 E7 Frontier Suite จะเปิดตัวผ่าน Cloud Solution Provider โดย Bundle ทั้ง M365 E5 Microsoft Entra Suite Microsoft 365 Copilot และ Microsoft Agent 365 รวมเป็นแพ็กเกจเดียว ซึ่งจะทำให้ Agentic AI กลายเป็นความสามารถพื้นฐานขององค์กรโดยอัตโนมัติครับ

Microsoft ประกาศชัดเจนว่า Prompt Injection และ Data Sovereignty กลายเป็นข้อกำหนดที่ต้องมีในการ Deploy AI สู่ Production โดยเฉพาะในอุตสาหกรรมที่มีการกำกับดูแลสูงอย่างธนาคารและภาครัฐ Azure AI Private Deployment Options กำลังกลายเป็น Criteria หลักในการเลือก Vendor ครับ

Azure AI Platform ปีนี้เน้นระบบ Model Orchestration แบบ Cooperative ครับ Small Language Model หรือ SLM ที่ Fine Tune แล้วจะจัดการงานปกติ และส่งต่องานที่ซับซ้อนให้ Foundation Model ขนาดใหญ่ทำต่อ วิธีนี้ทำให้ต้นทุนและ Latency ลดลงอย่างมีนัยสำคัญ เหมาะกับการ Scale AI ในองค์กรขนาดใหญ่ครับ

Microsoft ยังนำเสนอตัวอย่างองค์กรขนาดเล็กที่ใช้ Copilot Agent ดำเนินงานระดับโลกได้ในไม่กี่วัน สิ่งนี้เปิดโอกาสให้ธุรกิจขนาดกลางและเล็กสามารถแข่งขันกับองค์กรใหญ่ได้มากขึ้นครับ

มาสรุปห้าสิ่งที่ต้องจับตามองวันนี้กันครับ

หนึ่ง ประสิทธิภาพเหนือขนาด คือ Metric ใหม่ของ AI แล้วครับ Meta Muse Spark และงานวิจัยลดพลังงานหนึ่งร้อยเท่า กำลังเปลี่ยนเกณฑ์การแข่งขัน AI จากจำนวนพารามิเตอร์ มาเป็นประสิทธิภาพต่อการใช้ทรัพยากร

สอง ช่องว่างด้าน Governance ในธนาคารกำลังจะกลายเป็นระเบิดเวลา สามสิบเอ็ดเปอร์เซ็นต์ Deploy แต่มีเพียงสิบสองเปอร์เซ็นต์ที่มีกลยุทธ์ชัดเจน ผู้กำกับดูแลกำลังจับตาดูอยู่ครับ

สาม Microsoft Copilot Wave สาม และ E7 Bundle ที่เริ่มวันที่หนึ่งพฤษภาคม จะเปลี่ยนบทสนทนาขององค์กรจาก ควรซื้อ AI ไหม มาเป็น จะ Govern AI ที่มีอยู่แล้วอย่างไร ครับ

สี่ Databricks Genie GA ทำให้ Data Platform ก้าวข้ามจากการเป็นเพียง Storage และ Query Layer ไปสู่การเป็น Autonomous Operations Environment อย่างสมบูรณ์แล้วครับ

ห้า การลงทุนสามล้านล้านดอลลาร์ใน AI Infrastructure ภายในปีสองพันยี่สิบแปด บ่งบอกว่า AI ได้ข้ามจากซอฟต์แวร์องค์กร ไปสู่การเป็น Critical Infrastructure ในระดับเดียวกับพลังงานและโทรคมนาคมแล้วครับ

ขอบคุณที่รับฟัง MorningInsight ครับ หวังว่าข้อมูลวันนี้จะเป็นประโยชน์สำหรับการทำงานของทุกท่านนะครับ มีวันที่ดีและขับรถปลอดภัยนะครับ พบกันพรุ่งนี้เช้า
